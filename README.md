## 关于织游

织游 ZhiYou 是一款为热爱编织的人们打造的开源工具。只需上传一张图片，就能自动生成专业的编织图纸，让您的创意不再受限于数格子的繁琐。

> 「要是你能给我想办法打印出来纹样，尽量放大一点，没有我织不出来的！」
>
> —— 妈妈

小时候穿上妈妈给我织的立体恐龙毛衣，走在路上那种自豪是现在买什么衣服也没有获得过的。

现在，妈妈为我的餐厅织红酒杯带、法棍包……为我织我想要的，每一次都给我惊喜。但是我突然发现，妈妈为了我的想法手绘图样，每天在灯下对着A4打印纸，数格子，数针脚，其实并不简单。

于是，织游诞生了。为了妈妈，也为了每一个热爱编织的你。

## 功能特色

### 🎨 图片转编织图
上传任意图片，自动转换为编织图案。支持多种颜色数量和抖动模式，还原最真实的色彩效果。

### 🧶 多种针法支持
内置丰富的针法库，从基础的平针、反针到复杂的绞花、镂空，满足不同编织需求。

### 📐 自定义尺寸
根据您的成品尺寸和针距，自动计算图案大小，让每一件作品都恰到好处。

### 📄 PDF 导出
一键导出专业的编织图纸 PDF，包含：
- 封面页
- 针法图例
- 图案网格
- 编织说明

### 🌍 多语言支持
支持简体中文、繁体中文、English、日本語、한국어 五种语言。

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

打开浏览器访问 [http://localhost:3000](http://localhost:3000) 开始使用。

## 技术栈

- **Next.js 16** - React 框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式框架
- **Zustand** - 状态管理
- **Canvas 2D** - 高性能图案渲染
- **pdf-lib** - PDF 生成

## 开源协议

本项目基于 **AGPL-3.0** 协议开源。
